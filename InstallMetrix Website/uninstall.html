Uninstall

If you would like to uninstall InstallMetrix software, please follow the instructions below based on your operating system. It is simple and quick and can be done anytime. If you need more assistance, please visit the advertisers software page by clicking on the appropriate link.

 

Windows 7/ Windows 8

  1. 1. Click the start button
  2. 2. Click on control panel
  3. 3. Under programs, select Uninstall Program
  4. 4. Select the InstallMetrix enabled software you installed, and click the uninstall button above.
  5. 5. Follow the prompts and your done!

 

Windows Vista

  1. 1. Click the start button
  2. 2. Click on control panel
  3. 3. Under programs, select Uninstall Program
  4. 4. Select the InstallMetrix enabled software you installed, and click the uninstall button above.
  5. 5. Follow the prompts and your done!

 

Windows XP

  1. 1. Click the start button
  2. 2. Click on control panel
  3. 3. Click Add/Remove programs
  4. 4. Select the InstallMetrix enabled software you installed, and click the change/remove button.
  5. 5. Follow the prompts and your done!

 

Windows 95,98,ME,NT,2000

  1. 1. Click the start button
  2. 2. Click on control panel
  3. 3. Click Add/Remove programs
  4. 4. Select the InstallMetrix enabled software you installed, and click the change/remove button.
  5. 5. Follow the prompts and your done!
